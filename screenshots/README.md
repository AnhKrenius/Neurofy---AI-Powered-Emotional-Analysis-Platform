## Image Guidelines

- **Format**: PNG or JPG
- **Quality**: High resolution (at least 1440px width)
- **File size**: Optimize images (max 2MB per image)
- **Privacy**: Ensure no sensitive data is visible
- **Branding**: Include Neurofy branding where appropriate

## Adding Screenshots

1. Take screenshots of the live platform at www.neurofy.com.au
2. Ensure no personal/sensitive data is visible
3. Optimize images for web (use tools like TinyPNG)
4. Name files according to the list above
5. Add to this directory
